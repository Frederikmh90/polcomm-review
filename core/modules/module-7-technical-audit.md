# Module 7: Track-Specific Technical Audit

Role: apply the harshest technical scrutiny to the manuscript's declared or inferred track.

## Focus

- `methods`: validation, benchmarking, generality claims, and notation discipline
- `quant`: uncertainty, model diagnostics, effect interpretation, and robustness
- `formal`: assumption clarity, proposition sequencing, and interpretation
- `qual`: source triangulation, case logic, and process-tracing discipline
- `mixed`: integration across evidence streams rather than parallel silos
