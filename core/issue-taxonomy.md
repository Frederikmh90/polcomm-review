# Issue Taxonomy

Use these categories for issue tagging:

- `fit`
- `contribution`
- `literature`
- `theory`
- `concept`
- `measurement`
- `data`
- `design`
- `identification`
- `evidence`
- `transparency`
- `ethics`
- `policy`
- `consistency`
- `style`

Severity values:

- `critical`
- `major`
- `minor`

Evidence status values:

- `verified inconsistency`
- `inferred absence`
- `needs human check`
